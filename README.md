# ShortUrl
A short url webservice create in PHP.

## get short url
```
URL:http://x0a.top/t
MOTHOD:POST
FORM:JSON/POST_FORM
REQUEST:{
	"url":"*url"
}

RESPONSE:{
	"code":"*retCode",
	"url":"*url//shorted url",
	"qrcode":"inlineImage//shorted url image"
}
```
